# SQL-Fundamentals-Exercises
Conducting a company analysis for their employees salaries
Employee Database Analysis (SQL Project)

## Project Overview
This project focuses on analyzing an Employee Database using SQL to extract meaningful business insights. 

The dataset contains employee information such as:
- Employee ID
- Name
- Department
- Salary
- City

The goal of this project is to demonstrate how SQL can be used to perform data aggregation, grouping and business analysis to support decision-making.

## Objectives
The main objectives of this analysis are:

- Calculate total number of employees
- Determine total and average salaries
- Identify highest and lowest salaries
- Analyze salary distribution across departments
- Evaluate employee distribution by city
- Apply filtering using HAVING clauses for business rules

## SQL Concepts Covered

This project demonstrates practical use of:

- `COUNT()` → Total number of employees  
- `SUM()` → Total salary per department  
- `AVG()` → Average salary analysis  
- `MIN()` & `MAX()` → Salary range insights  
- `GROUP BY` → Department and city-level aggregation  
- `ORDER BY` → Sorting results for insights  
- `HAVING` → Filtering grouped data
